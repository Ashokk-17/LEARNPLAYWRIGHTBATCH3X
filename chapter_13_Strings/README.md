# Chapter 13 - Strings

This chapter covers JavaScript strings — creation, template literals, string properties, searching/checking, extracting substrings, transforming strings, and type conversion.

## Files

| File | Topic |
|---|---|
| [118_strings.js](118_strings.js) | Creating strings (quotes, template literals, multiline strings), `String()` conversion |
| [119_Strings_Properties.js](119_Strings_Properties.js) | `typeof`, `.length`, index access (`str[i]`, `.at()`), `.charAt()` / `.charCodeAt()` |
| [120_String_str_Check.js](120_String_str_Check.js) | Searching & checking: `.includes()`, `.startsWith()` / `.endsWith()`, `.indexOf()` / `.lastIndexOf()`, `.search()` with regex |
| [121_Substring.js](121_Substring.js) | Extracting substrings: `.slice()` vs `.substring()` |
| [122_Transform_str.js](122_Transform_str.js) | Transforming strings: case conversion, `.trim()`, `.replace()` / `.replaceAll()`, concatenation, `.split()` / `.join()` |
| [123_Str.js](123_Str.js) | String conversion: `.toString()`, `Number()`, `parseInt()`, `parseFloat()` |
